# 回防插件合集

## 内含插件

- FixHintColorMessages (字体颜色修复插件)

- MyWeaponAllocator ( !guns 选枪插件)

- retakes (回防主插件)

- retakes_standardallocator (回防标准配置插件)

- retakes_autoplant (自动安包插件)

- retakes_breaker (自动破窗破门插件)

- retakes_instadefuse (自动判断能否拆包并结束回合插件)

- retakes_sitepicker (可选择回防点位)
